# BuildingPlus
Building modification for Ultimate Chicken Horse, including Multi-Piece-Pickup, more Colors, no Building Restrictions and more!

## Multi Piece Cursor

The Cursor in Singleplayer Free Play sessions will be turned into a Multi Piece Cursor meaning you will be able to do the following:

- Marquee Selection of Placeables
- Pick up any Placeable of the Selection to pick up the whole Selection.
- Copy the Selection
- Delete all Placeables of the Selection
- Holding down LeftControl (configurable) to select multiple placeables without deselecting others or to de/select singles.
